The Industry Standard: Mini Sparkline

A tiny 64×20 line chart showing today's intraday price path. Color = net direction.

How Sparklines Capture Your Three Scenarios
A. Gap up, trend up all day
Open: 100 → drifted up to 108 (smooth uptrend)
AAPL 186.59 +8.0%
✓ Line climbs steadily = momentum all day
B. Gap down, trend down all day
Open: 100 → sold off to 92 (smooth downtrend)
CVNA 358.55 -8.0%
✓ Line falls steadily = distribution/selling all day
C. Dip then recovery (intraday reversal)
Open: 100 → dipped to 95 → recovered to 101
TSLA 245.30 +1.0%
✓ V-shape visible = bought the dip, recovered
D. Rangebound / sideways
Open: 100 → choppy between 98-102 → closed 100.50
GSAT 79.85 +0.5%
✓ Jagged sideways = indecision, no trend

Why Sparklines Work for Traders:

  • Shows the path, not just position: You see volatility, trend direction, reversals
  • Quick visual filter: Upward slope = buy pressure, downward = sell pressure
  • Pattern recognition: V-shape = dip bought, Λ-shape = rally rejected
  • Industry standard: Bloomberg, TradingView, Finviz all use this pattern

Implementation Notes:

  • Data: Use your existing intraday candles (1h or more granular if available)
  • Color: Green if close > open, Red if close < open, Gray if flat
  • Size: 64×20px is standard — fits in row without breaking layout
  • Rendering: SVG polyline — cheap to render, scales cleanly

This gives you trend + volatility at a glance before clicking the chart.
